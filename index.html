Java

Java คือ ภาษาโปรแกรมเชิงวัตถุ (Object Oriented Programming) ถูกสร้างขึ้นในปี 1991 และนำออกมาใช้สู่สาธารณะในปี 1995 พัฒนาโดย James Gosling และวิศวกรคนอื่น ๆ ของบริษัท Sun Microsystems (ปัจจุบัน Oracle Corporation เป็นเจ้าของ)

   ภาษา Java เดิมทีใช้ชื่อว่าภาษา Oak แต่ว่ามีปัญหาเรื่องลิขสิทธิ์จึงเปลี่ยนมาใช้ชื่อภาษา Java ที่มาจากกาแฟแทน ซึ่งเป้าหมายหลักของการสร้างภาษา Java มาคือเพื่อใช้แทนภาษา C++

“เพื่อสร้างภาษาโปรแกรมที่สามารถทำงานได้โดยไม่ต้องคำนึงถึง Hardware หรือ OS”

จุดเด่นของภาษา Java

  • การเขียนโปรแกรมเชิงวัตถุ (OOP)
    •   Java เป็นการเขียนโปรแกรมเชิงวัตถุหรือรูปแบบการเขียนโปรแกรมโดยการมองทุกอย่างให้เป็นวัตถุ สิ่งนี้ทำให้ง่ายต่อการจัดระเบียบและบำรุงรักษาโค้ดรวมถึงการนำโค้ดกลับมาใช้ซ้ำในแอปพลิเคชันต่าง ๆ

  • ใช้งานได้ทุกแพลตฟอร์ม (Platform Independence)
    •   โค้ดของภาษา Java สามารถใช้งานได้ทุกแพลตฟอร์มไม่ว่าจะเป็น Windows, iOS, Linux เพียงแค่ติดตั้ง JVM ทำให้เหมาะสำหรับการพัฒนาแอปพลิเคชันที่จำเป็นจะต้องทำงานบนหลาย ๆ แพลตฟอร์ม

  • มีชุดเครื่องมือและ Library ยอดนิยม (Popular Java Frameworks and Libraries)
    •   Java มีเครื่องมือและ Library ที่มากมายที่ช่วยให้เกิดความคล่องตัวในการเขียนโค้ดมากขึ้น โดยที่นักพัฒนา
    ไม่จำเป็นจะต้องเขียนโค้ดใหม่ตั้งแต่ต้น ตัวอย่างชุดเครื่องมือและ Library ยอดนิยม เช่น JavaServer Faces (JSF)
    และ JavaFX

  • ความปลอดภัย (Security)
    •   Java มีคุณสมบัติในการช่วยป้องกันภัยคุกคามและความปลอดภัยที่สูง เช่น Buffer Overflow และ SQL Injection จึงเหมาะที่จะพัฒนาแอปพลิเคชันที่มีข้อมูล ละเอียดอ่อนอย่างการทำธุรกรรมทางการเงิน นอกจากนี้ Java ยังถูกดีไซน์มาเพื่อการสื่อสารผ่านทางเครือข่ายระหว่างโทรศัพท์ไร้สาย ทำให้ Java มีระบบความปลอดภัยที่ดีมากยิ่งขึ้น
